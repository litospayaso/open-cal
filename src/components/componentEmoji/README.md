# ComponentEmoji

A simple component that displays an emoji based on the input text. It supports multiple languages (English, Spanish, French, German, Italian) and falls back to a pseudo-random emoji if no match is found.

## Usage

```html
<component-emoji text="apple"></component-emoji>
```

## Properties

| Property | Type | Description |
| Path   | Type     | Description                                           |
| ------ | -------- | ----------------------------------------------------- |
| `text` | `String` | The text input to match against the emoji dictionary. |

## Supported Languages

- English
- Spanish
- French
- German
- Italian
