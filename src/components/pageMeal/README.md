# PageMeal

The page for creating and editing meals.

## Features
- Allows users to create new meals or edit existing ones.
- Integrates with `PageSearch` to add food items to the meal.
- Supports saving meal drafts locally before persisting to the database.
- Provides functionality to save the final meal to the database.
- Allows adding the entire meal to the daily diary.

## URL Parameters
- `id`: The unique identifier of the meal to edit. Use `new` or omit to create a fresh meal.
